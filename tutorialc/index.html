Para começar: o que é a programação de um computador?

Programação é o processo de escrita, teste e manutenção de um programa de computador. O programa é escrito em uma linguagem de programação, embora seja possível, com alguma dificuldade, escrevê-lo diretamente em linguagem de máquina. Diferentes partes de um programa podem ser escritas em diferentes linguagens.

Ao iniciar o programa, deve-se declarar as variáveis como mandam as instruções do programa que você deseja montar.

Em C existem 4 tipos de variáveis que podem ser declaradas:
  • Caractere - char (a, b, c, 1, 2, #, ^, * )
  • Inteiro - int ( -2, -1, 0, 1, 2 )
  • Real - float ( 1.2, 2.5, -20.456 )
  • Lógico - int ( A variável lógica não existe em C, para representá-la usamos int com 0 para falso, e qualquer número inteiro para verdadeiro)

Operadores aritméticos em C fazem o computador realizar contas simples com os 4 operadores básicos. Os operadores aritméticos básicos são:
  • Adição - ( + )
  • Subtração - ( - )
  • Multiplicação - ( * )
  • Divisão - ( / )

Existem também os operadores relacionais, que são usados com os operandos e servem para definir expressões lógicas.

Os operadores relacionais são:
  • Igual - ( == )
  • Maior - ( > )
  • Menor - ( < )
  • Maior ou igual - ( >= )
  • Menor ou igual - ( <= )
  • Diferente - ( != )
